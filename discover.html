Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference between these three element manipulators is

  • innerHTML helps to get the content of the html and can set a new element in the html content.
  • innerText helps to create new text and also helps to get the text content from the html/
  • textContent is very similar to innerText and innerHTML it also set and change the content of the text in the element of the html.

Question-3: What is event delegation in the DOM?

Event delegation is dynamic way to handle event listing with the help of parent elements it dynamically changes the behavior of the child element by targeting it

Question-4: What is event bubbling in the DOM?

Event bubbling is DOM hierarchy term. when a event is triggered of an element in body, it first triggers itself and then goes through the previous elements to the top parent element. Hence causing the term "bubbling".

Question-5: How do you create, add, and remove elements using JavaScript?

You can Create element by document.createElement() and for adding element you can use .appendChild and lastly for removing any element you can use .remove and .removeChild.
